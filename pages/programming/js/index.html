準備

  1. 適当なところに「programming」という名前のフォルダを作ろう。
  2. そのフォルダ内で右クリックしてそのフォルダをvscodeで開こう。
  3. 「index.html」という名前のファイルを作ろう。
  4. テキストエディタ上で「!」を入力してEnterを押すとHTMLの雛型ができるはず。

    5. （<body>みたいな奴をタグというよ。
    タグはかっこみたいなもので、普通は開きと閉じがあるよ。 閉じは頭に/が付いているよ。）

  5. bodyタグの中にscriptタグを作ろう。JavaScriptはこのscriptタグの中に書いていくよ。
  6. 作ったindex.htmlをブラウザ（Chromeとか）で開こう。真っ白な画面が表示されるはず。
  7. F12を押して、コンソール画面を開こう。

JavaScriptの基本的な構文

簡単なプログラムを書けるようになろう。最初は全部呪文だと思ってコピペしていれば大丈夫だよ。

Hello World!

以下のコードを書いて、F5を押してページをリロードしよう。コンソールに「Hello World!」と表示されるはずだよ。

console.log("Hello World!")

「"」で囲まれた部分を文字列といって、文字を表示したいときはこれを使うんだ。 console.logでは文字列以外にもなんでも表示できるよ。 以下のコードを書いて、F5を押してページをリロードしよう。

console.log(123);
console.log(true);
console.log(false);

123は数値だよ。trueとfalseは真理値といって、正しいか正しくないかを表すよ。

プログラムは上から下へ実行されるよ。

変数宣言

変数とは数値や文字列などの値が入った箱のようなものだよ。

letの後に変数名、イコール、値の順番で書くことで変数を宣言できるよ。

宣言した変数は後で使うことができるよ。以下のコードを書いて、F5を押してページをリロードしよう。


let name = "ipota"; 
console.log(name);

変数は中身を書き換えることもできるよ。書き換える時はletは要らないよ。
以下のコードを書いて、F5を押してページをリロードしよう。 


let name = "ipota";
console.log(name);
name = "umeboshi";
console.log(name);

コメント

//を付ける事で一行だけ無視されるんだ。無視されている部分をコメントと呼ぶよ。以下のコードを書いて、F5を押してページをリロードしよう。

console.log(1);
// console.log(2);
console.log(3);

演算子

JavaScriptにはいくつかの演算子があるよ。以下のコードを書いて、F5を押してページをリロードしよう。

console.log(16 + 5); // 足し算
console.log(16 - 5); // 引き算
console.log(16 * 5); // 掛け算
console.log(16 / 5); // 割り算
console.log(16 % 5); // あまり

console.log(16 < 5); // 小なり
console.log(16 > 5); // 大なり
console.log(16 <= 5); // 小なりイコール
console.log(16 >= 5); // 大なりイコール
console.log(16 == 5); // イコール

console.log(true && false); // かつ
console.log(true || false); // または
console.log(!true); // 否定


let i = 0;

i += 1; // i = i + 1; の略記
console.log(i);

i -= 2; // i = i - 2; の略記
console.log(i);

i *= 3; // i = i * 3; の略記
console.log(i);

i /= 4; // i = i / 4; の略記
console.log(i);

let j = 0;

j++; // j = j + 1; の略記
console.log(j);

j--; // j = j - 1; の略記
console.log(j);

配列

配列というのは、いくつかの値の列の事だよ。以下のコードを書いて、F5を押してページをリロードしよう。


let array = ["a", "b", "c"];

// 後ろに[0]と付ける事で0番目の要素を取り出している。
console.log(array[0])
console.log(array[1])
console.log(array[2])

// このように書くと長さを取得できる
console.log(array.length)

関数

関数を使えばいくつかのプログラムをひとかたまりにして後で使えるようにできるよ。以下のコードを書いて、F5を押してページをリロードしよう。 


// 関数の宣言
// helloの部分は関数の名前
function hello() {
    console.log("hello");
    console.log("world");
}

// 関数の呼び出し
hello();
hello();
hello();

条件分岐

ここからは本当に呪文になってくるから、コピペして使えればそれでokだよ。

JavaScriptではある条件を満たしたときだけ実行するプログラムが作れるよ。以下のコードを書いて、F5を押してページをリロードしよう。 


if (true) {
    console.log("true");    
}

if (3 > 5) {
    console.log("3 > 5");
}

if (3 > 5) {
    console.log("3 > 5");
} else {
    console.log("3 <= 5");
}

let x = 10;

if (x < 3) {
    console.log("x < 3");
} else if (x < 5) {
    console.log("3 <= x < 5");
}

let y = 10;

if (y < 3) {
    console.log("y < 3");
} else if (y < 5) {
    console.log("3 <= y < 5");
} else {
    console.log("y >= 5");
}

繰り返し

繰り返し。以下のコードを書いて、F5を押してページをリロードしよう。


// 呪文だと思って覚えるかコピペする。
// 10回繰り返す。
for (let i = 0; i < 10; i++) {
    console.log(i);
}

// 配列の各要素を取り出して表示する。
let array = ["a", "b", "c"];
for (let i = 0; i < array.length; i++) {
    console.log(array[i]);
}
