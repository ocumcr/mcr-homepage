JavaScriptの基本的な構文

簡単なプログラムを書けるようになろう。最初は全部呪文だと思ってコピペしていれば大丈夫だよ。

Hello World!

下のコードを書いて、Ctrl+Sを押して保存して、ブラウザに戻ってF5を押してリロードしよう。コンソールに「Hello World!」と表示されるはずだよ。 


console.log("Hello World!")

「"」で囲まれた部分を文字列といって、文字を表示したいときはこれを使うんだ。 console.logでは文字列以外にもなんでも表示できるよ。 


console.log(123);
console.log(true);
console.log(false);

123は数値だよ。trueとfalseは真理値といって、正しいか正しくないかを表すよ。

プログラムは上から下へ実行されるよ。

変数宣言

変数とは数値や文字列などの値が入った箱のようなものだよ。

letの後に変数名、イコール、値の順番で書くことで変数を宣言できるよ。

宣言した変数は後で使うことができるよ。


let name = "ipota"; 
console.log(name);

変数は中身を書き換えることもできるよ。書き換える時はletは要らないよ。 


let name = "ipota";
console.log(name);
name = "umeboshi";
console.log(name);

コメント

//を付ける事で一行だけ無視されるんだ。無視されている部分をコメントと呼ぶよ。 


console.log(1);
// console.log(2);
console.log(3);

演算子

JavaScriptにはいくつかの演算子があるよ。


console.log(16 + 5); // 足し算
console.log(16 - 5); // 引き算
console.log(16 * 5); // 掛け算
console.log(16 / 5); // 割り算
console.log(16 % 5); // あまり

console.log(16 < 5); // 小なり
console.log(16 > 5); // 大なり
console.log(16 <= 5); // 小なりイコール
console.log(16 >= 5); // 大なりイコール
console.log(16 == 5); // イコール

console.log(true && false); // かつ
console.log(true || false); // または
console.log(!true); // 否定

JavaScriptにはいくつかの略記があるよ。


let i = 0;

i += 1; // i = i + 1; の略記
console.log(i);

i -= 2; // i = i - 2; の略記
console.log(i);

i *= 3; // i = i * 3; の略記
console.log(i);

i /= 4; // i = i / 4; の略記
console.log(i);

let j = 0;

j++; // j = j + 1; の略記
console.log(j);

j--; // j = j - 1; の略記
console.log(j);

配列

配列というのは、いくつかの値の列の事だよ。


let array = ["a", "b", "c"];

// プログラミングでは0から数え始めるんだ。注意しよう！
// 後ろに[0]と付ける事で0番目の要素を取り出している。
console.log(array[0])
console.log(array[1])
console.log(array[2])

// このように書くと長さを取得できる
console.log(array.length)

関数

関数を使えばいくつかのプログラムをひとかたまりにして後で使えるようにできるよ。


// 関数の宣言
// helloの部分は関数の名前
function hello() {
    console.log("hello");
    console.log("world");
}

// 関数の呼び出し
hello();
hello();
hello();

条件分岐

ここからは本当に呪文になってくるから、コピペして使えればそれでokだよ。

JavaScriptではある条件を満たしたときだけ実行するプログラムが作れるよ。


if (true) {
    console.log("true");    
}

if (3 > 5) {
    console.log("3 > 5");
}

if (3 > 5) {
    console.log("3 > 5");
} else {
    console.log("3 <= 5");
}

let x = 10;

if (x < 3) {
    console.log("x < 3");
} else if (x < 5) {
    console.log("3 <= x < 5");
}

let y = 10;

if (y < 3) {
    console.log("y < 3");
} else if (y < 5) {
    console.log("3 <= y < 5");
} else {
    console.log("y >= 5");
}

繰り返し

繰り返し。


// 呪文だと思って覚えるかコピペする。
// 10回繰り返す。
for (let i = 0; i < 10; i++) {
    console.log(i);
}

// 配列の各要素を取り出して表示する。
let array = ["a", "b", "c"];
for (let i = 0; i < array.length; i++) {
    console.log(array[i]);
}

FizzBuzzを作ってみよう！

FizzBuzzはプログラミングの定番練習問題だよ。

0から99までの数字を順番に表示するんだけど、

  • 3の倍数のときは数字の代わりに「Fizz」と表示する
  • 5の倍数のときは数字の代わりに「Buzz」と表示する
  • 3と5両方の倍数のときは「FizzBuzz」と表示する

というルールがあるよ。

ヒント：3の倍数かどうかは「3で割ったあまりが0かどうか」で判定できるよ。あまりは%で計算できるんだったね。

まずは自分で考えてみよう！

どうしても分からなかったら答えを見てね。

答えを見る 

for (let i = 0; i < 100; i++) {
    if (i % 3 == 0 && i % 5 == 0) { // i % 15 == 0でもいいよ
        console.log("FizzBuzz");
    } else if (i % 3 == 0) {
        console.log("Fizz");
    } else if (i % 5 == 0) {
        console.log("Buzz");
    } else {
        console.log(i);
    }
}

3と5の倍数の判定（FizzBuzz）を先に書かないといけないのがポイントだよ。なぜか考えてみよう。
